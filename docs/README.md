## 安装组件

使用 yarn 安装 AntDocs 主题：

```js
yarn add laihua-ui
```

或者使用 npm 安装它：

```js
npm i laihua-ui
```

如果你的网络环境不佳，推荐使用 [cnpm](https://github.com/cnpm/cnpm)。

::: warning
注意
请确保你的 Node.js 版本 >= 8.6。
:::
